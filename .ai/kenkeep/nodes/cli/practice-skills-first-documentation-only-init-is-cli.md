The public-facing documentation recommends the skill workflow (`/kk-curate`, `/kk-bootstrap`, `/kk-add`) for all curation and bootstrap operations. The `init` command is the only CLI command that is documented as a user-facing workflow. Other CLI commands are mentioned only in internal or contributor documentation where they describe the actual implementation machinery.
